Works / Projects

“推し活”サポート グッズ管理アプリ『 atsumeru 』

増え続ける“推し”のランダムグッズを効率的に管理・記録したいという自身の課題から開発したiOSアプリです。

【主な機能】グッズの登録、階層的データ管理、高度な絞り込み機能、検索機能、所持状況の画像生成、App内課金など

【工夫した点】大量のデータを効率的に管理し、柔軟なアクセスと軽快な動作を実現しました。また、ユーザーが直感的に操作できるよう、UI/UXには特にこだわりました。信頼性の高い課金システムの構築にも取り組みました。

【使用技術】UI: SwiftUI, Codable, JSON, FileManager, StoreKit 2, Google AdMob SDK (Rewarded Ads), async / await, Task, Git, GitHub, Xcode Simulator, TestFlight, Sandboxなど
